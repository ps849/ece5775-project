Instructions to synthesize the design.

1. run.tcl - This file has automated script to synthesize, simulate and cosimulate the design.
vivado_hls run.tcl - This command synthesizes the design and co-simulates the design.

2. run_bitstream.sh - This file has the automated script to generate the bitstream.
source run_bitstream.sh - Run this command to generate the bitstream after synthesizing the RTL design which is done in step 1.
